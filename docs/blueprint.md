# **App Name**: TaskZen

## Core Features:

- Task Management: Create, edit, and delete tasks with title, description, due date, category, and priority. Data is stored locally.
- Task Completion & Reminders: Mark tasks as complete/incomplete with visual distinction and local reminders/notifications for upcoming due dates.
- Task Organization: Filter, search, and sort tasks by category, status, or keyword.  Include drag-and-drop reordering.
- Dashboard & Settings: Display a dashboard with basic task completion statistics and a dark mode toggle that remembers the user's preference.

## Style Guidelines:

- Modern, minimal card-based layout for tasks.
- Primary color: Neutral gray for a clean look.
- Secondary color: Soft blue for task cards.
- Accent: Teal (#008080) for highlights and CTAs.
- Simple, consistent icons for categories and priority levels.
- Subtle animations for task completion and reordering.
